
tes3.getSound
========================================================

**Parameters:**

- `string`_ **id**:
    The id of the sound to look for.

**Returned:**

- `Sound`_ **result**: The `sound`_ object found, or ``nil`` if one wasn't found.

This function finds a sound by a given **id**.
