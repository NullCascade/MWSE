
tes3.isModActive
========================================================

**Parameters:**

- `string`_ **filename**:
    The filename of the mod to find, including the extension.

**Returned:**

- `boolean`_ **result**: ``true`` if the mod is active.

This function checks if a named mod is in the current load order.
